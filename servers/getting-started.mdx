## Adding Your Server

Get your server listed in just a few steps:

<Steps>
  <Step title="Navigate to Dashboard">
    Go to [rankly.live](https://rankly.live) and sign in with Discord
  </Step>
  
  <Step title="Add Your Server">
    Click "Add Server" from your dashboard
  </Step>
  
  <Step title="Grant Permissions">
    Select the server you want to list (you need Administrator or Manage Server permission)
  </Step>
  
  <Step title="Complete Information">
    Fill in your description, choose a category, add tags, and set up your server icon
  </Step>
  
  <Step title="Configure Features">
    Set up vote webhooks, customize your listing, and adjust settings
  </Step>
  
  <Step title="Submit & Launch">
    Submit your server for listing and start receiving votes
  </Step>
</Steps>

## Server Requirements

For optimal listing and to earn vote credits, meet these minimum requirements:

| Requirement | Minimum | Notes |
|-------------|---------|-------|
| **Server Age** | 12 hours | New servers can still list, but with limited visibility |
| **Member Count** | 250 members | Required to earn credits from votes |

<Tip>
Servers below these requirements will still appear in the directory! You'll gain full credit earning once you meet them.
</Tip>

## Common Tasks

### Setting Up Vote Webhooks

<CodeBlock language="javascript">
```javascript
const express = require('express');
const app = express();

app.use(express.json());

app.post('/webhook/votes', (req, res) => {
  const { userId, serverId, creditsAwarded } = req.body;

  if (creditsAwarded) {
    // Grant reward to user
    console.log(`User ${userId} voted for server ${serverId}`);
  }

  return res.status(200).json({ received: true });
});

app.listen(3000);
```
</CodeBlock>

## Troubleshooting

<Accordion title="Why isn't my server listed?">
Your server may be pending review or not meeting visibility requirements. Check that you have:
- At least 12 hours of server age
- Completed all required information
- A valid server icon or description
- No violations of Rankly policies
</Accordion>

<Accordion title="Why am I not earning credits?">
You need to meet minimum requirements:
- Server must be at least 12 hours old
- Must have at least 250 members
- Votes must be from verified users

Session credits also require adequate engagement metrics.
</Accordion>

<Accordion title="How do I withdraw earned credits?">
Navigate to your dashboard's transaction section and select "Withdraw." You can only withdraw earned credits (from votes), not session or purchased credits.
</Accordion>

<Accordion title="Can I have multiple servers listed?">
Yes! Add as many servers as you like to your account. Just ensure you have the proper permissions in each.
</Accordion>
